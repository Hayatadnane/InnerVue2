Optie 1: Gratis live met GitHubPages

Stap 1 GitHub-project maken

  1. Ga naar github.com en log in of maak een account.

  2. Klik op New repo (Nieuw repository).

  3. Geef een naam (bijv. humansight-site), kies voor Public, en klik Create repository.

Stap 2 HTML-bestand uploaden

  1. Ga naar je nieuwe repo.

  2. Klik op Add file > Upload file.

  3. Upload humansight-website.html.

  4. Klik op Commit changes.

Stap 3 Publiceren via GitHubPages

  1. Ga naar Settings > Pages.

  2. Kies als branch: main, map: /root (/), klik Save.

  3. Binnen enkele minuten staat je site live op een URL als:
    https://<gebruikersnaam>.github.io/humansight-site/

Optie 2: Vercel (ook gratis)

  1. Ga naar vercel.com en log in (via GitHub/Google).

  2. Klik op New Project.

  3. Koppel je GitHub-repo (zoals hierboven gemaakt).

  4. Standaardinstellingen zijn genoegzet project live.

  5. Je krijgt direct een .vercel.app-link zonder kosten.
